# 📌 Keen Keeper App

## 📝 Short Description
Keen Kepper is a Next.js based web application where users can track their interactions with friends such as Call, Text, and Video.  
All activities are displayed in a clean timeline format with filtering options for better user experience.

---

## 🛠️ Technologies Used

- Next.js (App Router)
- React.js
- JavaScript (ES6+)
- Tailwind CSS
- Context API

---

## ✨ Key Features

- 📅 **Activity Timeline System**  
  Displays friend interactions (Call, Text, Video) in chronological order.

- 🔍 **Filter Functionality**  
  Users can filter timeline logs by activity type or view all logs.

- 👥 **Dynamic Friend Integration**  
  Each activity is linked with specific friend data dynamically.
